  • dcterms:description "LinkedMDR stands for Linked Multimedia Document Representation.
    It represents the collective knowledge embedded in a heterogeneous document corpus of any domain-specific application.
    It is made of three main layers:
    (1) the core layer serving as a backbone and a mediator among the different layers;
    (2) the metadata layer linking descriptors of existing standards in metadata representation;
    (3) the pluggable domain-specific layer that can adapt to any domain-specific ontology.
    This version of LinkedMDR is based on descriptors from DC, TEI and MPEG-7 standards for the standardized metadata layer and an adapted version of the ifcOWL ontology for the pluggable domain-specific layer,"     @en
